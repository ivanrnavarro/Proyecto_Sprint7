# Proyecto_Sprint7
Repositorio del proyecto del Sprint 7 

A partir de un archivo de anuncios de vehículos en Estados Unidos, se ha construido una aplicación web que permita visualizar un histograma o un gráfico de dispersión para los odómetros de estos vehículos.

El usuario que utilice la aplicación podrá escoger entre realizar el histograma o el diagrama de dispersión.
